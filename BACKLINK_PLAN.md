# Backlink & Outreach Plan (dynamics-tim.dev)

## Ziel
Qualitativ hochwertige, thematisch relevante Backlinks aufbauen (Dynamics 365, Power Platform, Azure), ohne Linkfarmen. Fokus auf Expertise, Community-Beiträge und Partner-Verzeichnisse.

## Linkable Assets
- Power Platform ALM Checklist (HTML Guide) als verlinkbares Asset
- Klare Meta-/Schema-Daten und konsistente Canonicals für saubere Crawlability

## Outreach-Ideen (priorisiert)
1) Microsoft / Power Platform Communities mit praxisnahen Artikeln und Referenz-Link
2) Partner- und Tool-Verzeichnisse mit sauberem Profil
3) Gastbeiträge zu ALM, Governance, Integrationen
4) GitHub Repo Readme + Case-Notes mit Link zur Landingpage
5) Lokale IT-Meetups / User Groups (Vortrag + Ressourcenlink)
